# Сервис коротких ссылок + QR

Небольшой backend-сервис для сокращения ссылок и генерации QR-кодов.  
Проект выполнен как тестовое задание и оформлен в виде production-style приложения на базе **Yii2**.

## Стек

- Backend: PHP, Yii2 (Basic)
- Frontend: jQuery, Bootstrap
- DB: MySQL / MariaDB
- Инфраструктура: Docker, docker-compose

## Возможности

- Принимает URL на главной странице и отправляет запрос через Ajax.
- Валидирует URL (`http/https`) и проверяет доступность ресурса.
- Сохраняет ссылку в БД, генерирует короткую ссылку и QR-код.
- По короткой ссылке делает редирект на оригинальный URL.
- Логирует IP посетителя и User-Agent, увеличивает счётчик переходов.
- Обрабатывает ошибки и некорректные входные данные.

## Архитектурные и инженерные решения

- Разделение ответственности: контроллеры / сервисы / модели.
- Валидация входных данных на уровне DTO/форм.
- Генерация короткого кода с проверкой на коллизии.
- Логирование переходов (IP, User-Agent, timestamp).
- Подготовлена база для:
  - кеширования редиректов (например, через Redis),
  - rate limiting,
  - расширения API.
