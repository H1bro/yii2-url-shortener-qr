# Сервис коротких ссылок + QR

Тестовое задание на базе `Yii2 Basic + MySQL/MariaDB + jQuery + Bootstrap`.

## Что умеет сервис

- Принимает URL на главной странице и отправляет запрос через Ajax.
- Валидирует URL (`http/https`) и проверяет доступность ресурса.
- Сохраняет ссылку в БД, генерирует короткую ссылку и QR-код.
- По короткой ссылке делает редирект на оригинальный URL.
- Логирует IP посетителя и user-agent, увеличивает счетчик переходов.

## Быстрый старт

Запуск в Docker:

```bash
docker compose up --build -d
```

Полная инструкция развертывания и проверки:

- [`docs/DEPLOY_AND_RUN.md`](docs/DEPLOY_AND_RUN.md)

## Основные роуты

- `/` — форма генерации короткой ссылки.
- `/api/shorten` — Ajax эндпоинт создания короткой ссылки.
- `/r/<code>` — переход по короткой ссылке с логированием.
